# Employee_Profile_Generator

This is a simple Python project I did, trying to implement **String Manipulation**, specifically on the following areas:
  1. **String Concatenation** - through use of the '+' operator.
  2. **String Interpolation** - through use of Python's _f-string_ functionality.
  3. **String slicing** - by use of the '[:]' that implements the  _string indexing_ functionality to define the start and stop points.

All you need to do is run the file on a terminal to get the output.
